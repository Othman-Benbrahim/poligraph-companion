# poligraph-companion
Extension navigateur (Firefox, portable Chrome) : le portail citoyen sur les politiques français. Recherche instantanée, fiches complètes (biographie, mandat, contact institutionnel, activité parlementaire, affaires judiciaires, fact-checks, votes, déclarations HATVP), fonctionnement hors ligne, et assistant IA optionnel en BYOK.
