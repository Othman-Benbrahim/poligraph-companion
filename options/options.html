Poligraph Companion — Assistant IA

L'assistant fonctionne en BYOK : vous fournissez votre propre clé (ou un modèle local via Ollama / LM Studio, sans clé). Elle est stockée localement dans votre navigateur et n'est envoyée qu'au fournisseur choisi.

Les appels aux endpoints distants nécessitent une autorisation d'accès que le navigateur demandera à l'enregistrement. Pour Ollama, lancez-le avec OLLAMA_ORIGINS=moz-extension://* ollama serve (Firefox) afin d'autoriser l'extension.
